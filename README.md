## Stack

- **React 18** + **TypeScript** + **Vite**
- **Tailwind CSS** — estilização utility-first
- **Framer Motion** — animações e scroll reveal
- **React Router DOM** — roteamento
- **Lucide React** — ícones
- **EmailJS** — envio de emails pelo formulário de contato

---

## Rodando localmente

```bash
# 1. Instalar dependências
npm install

# 2. Configurar variáveis de ambiente
cp .env.example .env.local
# Editar .env.local com suas chaves do EmailJS

# 3. Iniciar servidor de desenvolvimento
npm run dev
```

## Variáveis de ambiente

Crie um arquivo `.env.local` na raiz com base no `.env.example`:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

As chaves são obtidas no painel do EmailJS.
